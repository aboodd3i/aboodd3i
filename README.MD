## 🔥 MORE PROJECTS

### Automated Parking Allocation System 🚗
**[GitHub Repo](https://github.com/aboodd3i/Parking-Allocation-System)**

A CLI-based parking management system built in **C (Linux environment)** using the **ncurses library**.  
The system simulates real-world parking allocation by prioritizing vehicles based on priority levels and waiting time.

To ensure fairness, it implements an **aging mechanism** to prevent starvation of low-priority vehicles.
